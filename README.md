Students: Mark Yeriskin, Lesbay Bolatkhan, Amambay Nurzhan

The application is a web service for tracking personal income and expenses, where the user can register, log in via JWT, add transactions by amount, category, and type (income/expense), view general statistics (total income, expenses, and balance), and delete transactions—all of this is implemented in Angular using services, forms, routing, and HTTP requests to the Django REST API, which provides full backend logic with models, serializers, authorization, and CRUD operations.
